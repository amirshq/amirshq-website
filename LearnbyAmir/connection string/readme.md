## Connection String Reference

- **Relational SQL**: PostgreSQL, MySQL/MariaDB, SQL Server, SQLite, Oracle, plus cloud providers (Supabase, Neon, PlanetScale).
- **NoSQL**: MongoDB, Azure Cosmos DB, Firebase/Firestore, DynamoDB.
- **Vector & Graph**: ChromaDB, Pinecone, Weaviate, Neo4j.
- **Cloud & Services**: Azure (Blob, Service Bus, Event Hubs, SQL, OpenAI, AI Search), AWS (S3, SQS/SNS, RDS).
- **Messaging & Cache**: RabbitMQ, Kafka, NATS, Redis, Memcached.
- **Network & Auth**: FTP/SFTP, SMTP/IMAP, LDAP/AD, WebSocket, API keys, OAuth2/JWT, plus security best practices.

Open `connectionstring.html` in your browser to browse categories, copy examples, and view tips/warnings.
