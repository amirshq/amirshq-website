## Dialog CX

Dialog CX is Google’s low-code conversational AI platform built on Google Cloud, used to design, build, and manage chatbots and voicebots.

- **`dialogcx.html`**: Visual, interactive guide to Dialog CX concepts, best practices, and implementation patterns.
- **`dialogcxflowchart.html`**: Flowchart-style view of conversation design, intents, and fulfillment paths.

Open either HTML file directly in your browser to explore the content and reuse patterns in your own Dialog CX projects.
